annotations-processor
==============

Código demonstra através de uma API alto nível de reflexão, como ler e utilizar metadados em Java.
O exemplo usa uma anotação para determinar conversores. O papel do conversor é transformar o conteúdo de um bean em um conteúdo de tipo genérico.

Conheça a Reflections API em https://code.google.com/p/reflections/
